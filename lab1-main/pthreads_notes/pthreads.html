Pthreads Introduction

Pthreads

Pthreads is a user-level, preemptive kernel running inside a UNIX process. Pthreads provides process management, synchronization, and communication functions. It also provides memory allocation services. Pthreads processes are scheduled using a three priority, round-robin, preemptive scheduling algorithm. No process is allowed to run if there are any ready processes of higher priority.

  • Process Management
  • Process Synchronization
  • Interprocess Communication
  • Memory Allocation
  • Sleep
  • Timers
