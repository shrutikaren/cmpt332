Provided Files

There is only 1 provided file for this lab sim.c. This contains a basic memory access simulator with a random page replacement algorithm. You may work in groups of at most 2 students. Each student must supply their own solution.

The pagerep-sim.c depends on the math library. The simulator also has 2 mandatory parameters and 1 optional parameter. The mandatory parameter are npages which states the number of pages to use and nframes which states the number of frames to use for simulating main memory. The optional parameter times is used to run the main loop for page referencing a specific number of times, this disables the sleeping between prints.

Description

In this lab, you will be implementing the second chance page replacement algorithm to choose the victim page for this simulator. The reference and dirty bits are automatically set when a page is accessed. You will be implementing the algorithm in find_victim_slot().
